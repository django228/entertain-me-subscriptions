## Задача

Имеется страница менеджера подписки для сайтов.
Макет: https://www.figma.com/design/qk27QeILypdh2adAhrVWwZ

В чем суть:

Есть несколько сайтов, нужно дать возможность подписываться и отписываться от нескольких сайтов одновременно, то есть менять состояние подписки с помощью переключателей. 
Чем больше переключателей в положении "вкл", тем больше радости в жизни человека; fun - этот показатель отображается в левой части экрана.
Можно, также, пренебречь всеми переключателями и нажать на кнопку отписаться от всего сразу.
В макете имеется всплывающие окно (попап), которые появляются при нажатии кнопку "Unsubscribe from all" и отображают разницу между старым и новым значением.

Задание:
1. Адаптивно сверстать страницу (вид мобильной версии представить на свое усмотрение);
2. Предусмотреть изменение прогресс бара (и процентное значение) FUN YOU GET, в зависимости от количества включенных подписок; (если вы сделаете анимацию цифр, это будет большим плюсом)
3. Предусмотреть изменение статуса Subscrubed;
4. При нажатии на кнопку "Unsubscribe from all" выключить все подписки;

Страница должна получать данные из JSON файла Subscribe.json.
При нажатии на кнопки "Subscribe" и "Unsubscribe from all" страница должна формировать информацию о текущем состоянии подписок и выводить ее в json формате (соответствующему файлу subscribe.json) в консоль браузера.
Также выводить попап с разницей между старым и новым значением FUN YOU GET.

!json заполнен примерно, вы можете изменять его как вам удобно
